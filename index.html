📈 Why Bother?

Every day, most of us check the same things: weather, news, crypto, stocks. That adds up to a dozen taps across five apps before lunch.

This automation condenses all of that into a single email that arrives automatically at the same time every day. No notifications, no doomscrolling, no decisions. Just the information you actually need:

  • 🌦 Weather — Should I bring an umbrella today?
  • 💬 Quote — A small dose of perspective.
  • 📺 Headlines — What's happening in the world.
  • 💰 Crypto & Stocks — How are my investments doing?

It's five minutes to set up and then you never think about it again. It just shows up.

🛠️ Setup: 5 Minutes, 3 Steps

You need a free GitHub account and a Gmail address. That's it.

  1. 1
    Generate a Gmail App Password

    Go to Google App Passwords, select Mail, click Generate, and copy the 16-character password. (Requires 2-Step Verification enabled on your Google account.)

  2. 2
    Add secrets to your GitHub repo

    In your repository on GitHub, go to Settings → Secrets and variables → Actions and add:

    SENDER_EMAILYour Gmail address
    SENDER_PASSWORDThe app password from step 1
    RECEIVER_EMAILWhere you want the email sent
    TWELVEDATA_API_KEY(Optional) Twelve Data key for PH stocks
  3. 3
    Push the code to GitHub

    Push the repo. The workflow activates automatically and sends your first email at 2PM PH time. You can also trigger it manually from the Actions tab.

✅ That's it. No server setup, no cron config, no monthly fees. Your daily brief will arrive automatically from tomorrow afternoon.

🛡️ What if Something Breaks?

  • If an API is down, the email still sends — that section just shows a friendly "unavailable" message.
  • If the email fails to send, the content is saved to a fallback file in the repo so nothing is lost.
  • If the entire workflow fails, a push notification is sent via ntfy.sh so you know immediately.

🎨 Make It Yours

Everything is configurable by editing the Config block at the top of send_email.py:

# Change the city and coordinates:
lat: float = 11.6083       # Borongan City
lon: float = 125.4358
city: str = "Borongan City, Eastern Samar"

# Change the timezone:
timezone: str = "Asia/Manila"

# Or run locally to test:
python send_email.py --dry-run   # Preview the email
python send_email.py --local     # Send using .env file
